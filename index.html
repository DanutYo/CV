How to install Browser Sync

Browser Sync is a tool that help you to see in real time what changes do you make at your code, this is really useful.

  1. First of all install Node from Node website.

  2. Second move is to install Browser-sync with command "npm install -g browser-sync"

  3. Then if you want to start browser sync must to write in cmd "browser-sync start --server -- files "*""
